1.1 Problem Statement

Create a Python program to calculate the customer’s bill in a cafe.


1.2 Inputs
	•	Customer name
	•	Coffee quantity
	•	Tea quantity
	•	Sandwich quantity


1.3 Outputs
	•	Customer name
	•	Coffee quantity
	•	Tea quantity
	•	Sandwich quantity
	•	Total price


1.4 Typical Process Flow
	1.	Enter customer name.
	2.	Enter coffee quantity.
	3.	Enter tea quantity.
	4.	Enter sandwich quantity.
	5.	Calculate the total.
	6.	Show the receipt.


1.5 Constraints
	•	Coffee = RM 8.50
	•	Tea = RM 6.00
	•	Sandwich = RM 12.00

⸻

2. Break the problem into smaller tasks
	1.	Get customer name.
	2.	Get coffee quantity.
	3.	Get tea quantity.
	4.	Get sandwich quantity.
	5.	Calculate total price.
	6.	Print receipt.

3. pseudocode

    Start

Set coffee_price = 8.50
Set tea_price = 6.00
Set sandwich_price = 12.00

Input customer_name
Input coffee
Input tea
Input sandwich

total = (coffee × coffee_price) +
        (tea × tea_price) +
        (sandwich × sandwich_price)

Print customer_name
Print coffee
Print tea
Print sandwich
Print total

End
